IOT Project - BUS SCHEDULING SYSTEM



Why choose this project :- 

&#x09;--> So IITG has a scheduled bus timings and people go on those scheduled bus timinigs and Busses run on these scheduled times whether they have enough passengers or not. So If enough passengers are not there then the bus will not efficiently run (Petrol,deisel waste) which leads to carbon emission much. So To solve this We propose a Green IOT themed - BUS SCHEDULING SYSTEM.



what are we doing in this project :- 

1. Sensing and Publishing data using Dynamic\_QR + MQTT
2. Creating a Dashboard to see live passengers count
3. Creating a Intelligent model by taking this Data and Trying to predict future Passeneger counts with LSTM model
